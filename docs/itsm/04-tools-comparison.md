## 1. 주요 ITSM 도구 개요

| 도구 | 대상 규모 | 핵심 강점 | 약점 | 가격대 (3년/50에이전트 기준) |
|------|-----------|-----------|------|---------------------------|
| **ServiceNow** | 대기업 | 최대 기능, 강력한 자동화/확장성 | 높은 비용, 복잡한 구현 | $800K ~ $1.5M+ |
| **Jira Service Management** | 중견/DevOps | Atlassian 생태계 통합, DevOps 친화적 | 대규모 ITIL 프로세스 약함 | $120K ~ $250K |
| **BMC Helix (Remedy)** | 대기업 | 강력한 ITIL 정렬, 하이브리드 배포 | 레거시 UI, 높은 구현 비용 | $600K ~ $1.2M |
| **Freshservice** | 중소/중견 | 빠른 도입, 직관적 UX, AI 기능 | 대기업 규모 한계 | $90K ~ $175K |
| **Zendesk** | 중소기업 | 옴니채널 지원, 사용 편의성 | ITIL 프로세스 깊이 부족 | $75K ~ $150K |
| **ManageEngine ServiceDesk Plus** | 중소/중견 | 합리적 가격, 온프레미스/클라우드 | 글로벌 지원 상대적 약함 | $50K ~ $120K |

## 2. 도구별 상세 분석

### ServiceNow

| 항목 | 내용 |
|------|------|
| **공급사** | ServiceNow, Inc. (미국) |
| **배포 방식** | SaaS (클라우드 전용) |
| **핵심 모듈** | ITSM, ITOM, ITAM, HR Service Delivery, SecOps, CSM |
| **자동화** | Flow Designer, IntegrationHub, Now Assist (생성형 AI) |
| **통합** | 1,200+ 사전 구축 통합, REST/SOAP API |
| **ITIL 정렬** | ITIL 4 인증 (19개 실천), 업계 최고 수준 |
| **강점** | 단일 플랫폼 전략, 강력한 워크플로우 자동화, 엔터프라이즈 확장성 |
| **약점** | 높은 TCO, 긴 구현 기간(6~18개월), 전문 인력 필요 |
| **적합 조직** | 직원 5,000명 이상, 복잡한 ITSM 요구사항, 대규모 자동화 필요 |

### Jira Service Management (JSM)

| 항목 | 내용 |
|------|------|
| **공급사** | Atlassian (호주) |
| **배포 방식** | 클라우드, Data Center (온프레미스) |
| **핵심 모듈** | 서비스 요청, 인시던트, 문제, 변경, 자산/구성 관리 |
| **자동화** | Jira Automation, Opsgenie 통합 (on-call 관리) |
| **통합** | Confluence, Bitbucket, Jira Software, 1,000+ Marketplace 앱 |
| **ITIL 정렬** | ITIL 4 주요 실천 지원, DevOps/Agile 통합에 강점 |
| **강점** | 개발-운영 통합, Atlassian 생태계, 합리적 가격 |
| **약점** | 복잡한 ITIL 프로세스 지원 제한, 대규모 확장성 |
| **적합 조직** | DevOps 문화, Atlassian 도구 사용 중, 개발-운영 통합 필요 |

### BMC Helix (Remedy)

| 항목 | 내용 |
|------|------|
| **공급사** | BMC Software (미국) |
| **배포 방식** | SaaS, 온프레미스, 하이브리드 |
| **핵심 모듈** | ITSM, Digital Workplace, Multi-Cloud Management, Discovery |
| **자동화** | BMC Helix Operations Management, AI/ML 기반 자동화 |
| **통합** | CMDB 중심 통합, REST API, 다양한 인프라 연동 |
| **ITIL 정렬** | ITIL 4 인증, 20년+ ITIL 경험으로 깊은 프로세스 지원 |
| **강점** | 하이브리드 배포 옵션, 강력한 CMDB, 엔터프라이즈 성숙도 |
| **약점** | 레거시 UI(개선 중), 높은 구현/운영 비용, 가파른 학습 곡선 |
| **적합 조직** | 온프레미스 요구, 기존 Remedy 사용 조직, 복잡한 인프라 |

### Freshservice

| 항목 | 내용 |
|------|------|
| **공급사** | Freshworks (인도/미국) |
| **배포 방식** | SaaS (클라우드 전용) |
| **핵심 모듈** | 인시던트, 서비스 요청, 변경, 문제, 자산, 프로젝트 관리 |
| **자동화** | Workflow Automator, Freddy AI (가상 에이전트) |
| **통합** | Freshworks 생태계, 200+ 사전 통합, API |
| **ITIL 정렬** | ITIL 4 주요 실천 지원, PinkVERIFY 인증 |
| **강점** | 빠른 도입(2~4주), 직관적 UI/UX, AI 기반 자동화 |
| **약점** | 대기업 규모 확장 제한, 고급 커스터마이징 한계 |
| **적합 조직** | 중소/중견기업, 빠른 도입 필요, 사용자 경험 중시 |

### Zendesk

| 항목 | 내용 |
|------|------|
| **공급사** | Zendesk (미국) |
| **배포 방식** | SaaS (클라우드 전용) |
| **핵심 모듈** | 티켓 관리, 옴니채널 지원, 셀프서비스, 분석 |
| **자동화** | 트리거/자동화 규칙, Answer Bot (AI) |
| **통합** | 1,500+ Marketplace 앱, REST API |
| **ITIL 정렬** | 기본적 ITSM 프로세스 지원 (전통적 ITIL보다 CX 중심) |
| **강점** | 옴니채널 통합, 쉬운 설정, 고객 경험 중심 |
| **약점** | ITIL 프로세스 깊이 부족, CMDB/변경관리 약함 |
| **적합 조직** | 고객 지원 중심 IT 조직, 중소기업, CX 우선 |

### ManageEngine ServiceDesk Plus

| 항목 | 내용 |
|------|------|
| **공급사** | Zoho Corporation (인도) |
| **배포 방식** | 온프레미스, 클라우드 |
| **핵심 모듈** | 인시던트, 문제, 변경, 자산, CMDB, 프로젝트 관리 |
| **자동화** | 비즈니스 규칙, SLA 자동화, 커스텀 스크립트 |
| **통합** | ManageEngine 제품군, REST API, Active Directory |
| **ITIL 정렬** | PinkVERIFY 인증 (주요 ITIL 프로세스) |
| **강점** | 합리적 가격, 온프레미스 옵션, ManageEngine 생태계 |
| **약점** | UI 현대화 필요, 글로벌 지원 인프라, AI 기능 상대적 약함 |
| **적합 조직** | 예산 제약 중소/중견기업, 온프레미스 선호, ManageEngine 사용 중 |

## 4. 도구 선택 가이드

### 조직 규모별 권장

| 조직 규모 | 1순위 추천 | 2순위 추천 | 이유 |
|-----------|-----------|-----------|------|
| 스타트업 (1~50명) | Freshservice Starter | Zendesk | 빠른 도입, 낮은 비용 |
| 중소기업 (50~500명) | Freshservice | ManageEngine | 가성비, 직관적 UX |
| 중견기업 (500~2,000명) | Jira Service Management | Freshservice | DevOps 통합, 합리적 비용 |
| 대기업 (2,000~10,000명) | ServiceNow | BMC Helix | 확장성, 엔터프라이즈 기능 |
| 초대기업 (10,000명+) | ServiceNow | BMC Helix | 글로벌 표준, 복잡한 요구 대응 |

### 요구사항별 권장

| 핵심 요구사항 | 권장 도구 | 이유 |
|---------------|-----------|------|
| DevOps/Agile 통합 | Jira Service Management | Atlassian 생태계, 개발-운영 연계 |
| 최대 ITIL 정렬 | ServiceNow, BMC Helix | 가장 포괄적인 ITIL 프로세스 지원 |
| 빠른 도입 (1개월 이내) | Freshservice, Zendesk | 직관적 설정, SaaS 기반 |
| 온프레미스 필수 | BMC Helix, ManageEngine | 온프레미스 배포 옵션 제공 |
| 예산 제약 | ManageEngine, Freshservice | 합리적 라이선스 비용 |
| 고객 경험 중심 | Zendesk | 옴니채널 CX에 특화 |
| AI/자동화 우선 | ServiceNow, Freshservice | Now Assist, Freddy AI |
| 규정 준수/감사 | ServiceNow, BMC Helix | GRC 모듈, 감사 추적 |

## 5. 도입 시 고려사항

### 평가 체크리스트

| 평가 항목 | 확인 사항 | 중요도 |
|-----------|-----------|--------|
| 기능 적합성 | 현재/미래 ITSM 프로세스 지원 여부 | 필수 |
| 통합 역량 | 기존 시스템(모니터링, CMDB, AD)과 연동 | 필수 |
| 확장성 | 조직 성장에 따른 확장 가능성 | 높음 |
| TCO (총소유비용) | 라이선스 + 구현 + 운영 + 교육 비용 | 높음 |
| 벤더 안정성 | 재무 상태, 시장 위치, 로드맵 | 높음 |
| 사용자 경험 | 최종 사용자/에이전트 UI 직관성 | 중간 |
| 보안/컴플라이언스 | SOC 2, ISO 27001, GDPR 준수 | 필수 |
| 도입 기간 | 파일럿 → 전사 확대 소요 시간 | 중간 |
| 벤더 지원 | 한국어 지원, 현지 파트너 유무 | 중간 |
| 커뮤니티/생태계 | 사용자 커뮤니티, 마켓플레이스, 문서 | 낮음~중간 |

### TCO 구성요소

| 비용 항목 | 설명 | 비중 |
|-----------|------|------|
| 라이선스/구독료 | 에이전트 수, 티어별 과금 | 30~40% |
| 구현/컨설팅 | 초기 설정, 커스터마이징, 데이터 마이그레이션 | 20~30% |
| 교육 비용 | 관리자, 에이전트, 최종 사용자 교육 | 5~10% |
| 운영/유지보수 | 지속적 관리, 업그레이드, 최적화 | 15~25% |
| 통합 비용 | 타 시스템 연동 개발/유지 | 5~15% |
