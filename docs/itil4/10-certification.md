## 인증 체계 개요

ITIL 4 인증은 PeopleCert(구 Axelos)에서 관리하며, 단계별로 IT 서비스 관리 전문성을 검증합니다.

```
                    ┌─────────────┐
                    │   Master    │
                    └──────┬──────┘
                           │
              ┌────────────┴────────────┐
              │                         │
     ┌────────┴────────┐     ┌─────────┴────────┐
     │ Managing        │     │ Strategic         │
     │ Professional    │     │ Leader            │
     │ (MP)            │     │ (SL)              │
     └────────┬────────┘     └─────────┬─────────┘
              │                        │
     ┌────────┴────────┐     ┌─────────┴─────────┐
     │ CDS, DSV, HVIT  │     │      DITS          │
     │      + DPI      │     │      + DPI          │
     └────────┬────────┘     └─────────┬──────────┘
              │                        │
              └────────────┬───────────┘
                    ┌──────┴──────┐
                    │ Foundation  │
                    └─────────────┘
```

## Foundation

| 항목 | 내용 |
|------|------|
| **대상** | ITIL 입문자, IT 서비스 관리에 관심 있는 모든 직군 |
| **전제 조건** | 없음 |
| **학습 내용** | ITIL 4 핵심 개념, 7대 지도 원칙, SVS, 서비스 가치 사슬, 4가지 차원, 34개 프랙티스 개요 |
| **시험 형식** | 객관식 40문항, 60분 |
| **합격 기준** | 65% (26/40 정답) |
| **시험 방식** | 온라인 또는 시험 센터 |
| **유효 기간** | 만료 없음 (평생 유효) |

### Foundation 시험 준비 팁

- 공식 교재 (ITIL 4 Foundation) 정독
- 7대 지도 원칙과 34개 프랙티스 명칭/목적 암기
- SVS 구성요소와 서비스 가치 사슬 6가지 활동 이해
- 모의 시험을 통한 반복 학습

## Managing Professional (MP)

IT 서비스 운영 실무자를 위한 트랙으로, 4개 모듈을 모두 이수해야 MP 자격을 취득합니다.

### MP 모듈 상세

| 모듈 | 정식 명칭 | 핵심 내용 |
|------|----------|-----------|
| **CDS** | Create, Deliver and Support | 서비스 가치 흐름 설계, 서비스 생성/전달/지원, 팀 문화, 통합/자동화 |
| **DSV** | Drive Stakeholder Value | 고객 여정 매핑, 이해관계자 관계 관리, 수요 형성, 서비스 제공/온보딩 |
| **HVIT** | High-velocity IT | 디지털 조직의 고속 IT 운영, Agile/Lean/DevOps 통합, 디지털 제품 관리 |
| **DPI** | Direct, Plan and Improve | 전략적 방향 설정, 거버넌스, 지속적 개선, 조직 변화 관리 (MP/SL 공통) |

### MP 각 모듈 시험 정보

| 항목 | 내용 |
|------|------|
| **전제 조건** | ITIL 4 Foundation 보유 |
| **시험 형식** | 객관식 40문항, 90분 |
| **합격 기준** | 70% (28/40 정답) |
| **오픈 북** | 아니오 |

---

## Strategic Leader (SL)

IT 전략과 비즈니스 목표를 연계하는 리더를 위한 트랙으로, 2개 모듈을 이수해야 합니다.

### SL 모듈 상세

| 모듈 | 정식 명칭 | 핵심 내용 |
|------|----------|-----------|
| **DPI** | Direct, Plan and Improve | MP 트랙과 공통 모듈 (위 참조) |
| **DITS** | Digital and IT Strategy | 디지털 전략 수립, IT 전략과 비즈니스 전략 정렬, 디지털 혁신, 리스크/기회 관리 |

### DITS 시험 정보

| 항목 | 내용 |
|------|------|
| **전제 조건** | ITIL 4 Foundation 보유 |
| **시험 형식** | 객관식 40문항, 90분 |
| **합격 기준** | 70% (28/40 정답) |
| **특이사항** | 케이스 스터디 기반 문제 포함 |

---

## ITIL Master

| 항목 | 내용 |
|------|------|
| **전제 조건** | MP + SL 모두 취득 |
| **추가 요건** | 최소 5년 이상의 IT 서비스 관리 실무 경험 |
| **평가 방식** | 실무 프로젝트 제출 및 인터뷰 (시험 없음) |
| **목적** | ITIL을 실제 비즈니스 환경에 적용하고 개선한 경험 검증 |
| **취득 난이도** | 매우 높음 (전 세계 소수만 보유) |

## 인증 경로 요약

| 레벨 | 명칭 | 모듈 수 | 대상 |
|------|------|---------|------|
| 1 | **Foundation** | 1개 | 입문자, 모든 IT 직군 |
| 2a | **Managing Professional** | 4개 (CDS+DSV+HVIT+DPI) | IT 실무자, 관리자 |
| 2b | **Strategic Leader** | 2개 (DPI+DITS) | IT 전략가, CIO, CTO |
| 3 | **Master** | MP+SL+실무경험 | 최고 전문가 |

### 추천 경로

- **IT 운영/서비스 데스크 담당자**: Foundation → CDS → DSV → HVIT → DPI (MP)
- **IT 관리자/리더**: Foundation → DPI → DITS (SL)
- **풀스택 전문가**: Foundation → MP + SL → Master

## 시험 정보

### 공통 사항

| 항목 | 내용 |
|------|------|
| **인증 기관** | PeopleCert |
| **시험 언어** | 영어, 일본어, 중국어, 독일어, 스페인어 등 (한국어는 Foundation만 지원) |
| **시험 방식** | 온라인 감독(Online Proctored) 또는 시험 센터 |
| **결과 통보** | 시험 완료 후 즉시 (온라인) |
| **재시험** | 제한 없음 (별도 비용 발생) |

### 비용 (2026년 기준, 대략적)

| 인증 | 시험 비용 (USD) |
|------|----------------|
| Foundation | ~$380 |
| CDS / DSV / HVIT / DPI | 각 ~$480 |
| DITS | ~$530 |
| Master | ~$3,000+ (심사 비용 포함) |

> 교육 과정 비용은 별도이며, 공인 교육 기관(ATO)을 통해 수강 가능합니다.
